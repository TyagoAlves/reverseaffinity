15+ Ferramentas Profissionais

Move, Brush, Eraser, Pen (Bezier), Gradient, Shape, Text, Clone Stamp, Healing Brush, Crop, e mais. Cada ferramenta com atalho de tecla única.

Camadas + Blend Modes

12 modos de mesclagem (Normal, Multiply, Screen, Overlay, etc.), opacidade, visibilidade, máscaras e grupos. Tudo não-destrutivo.

Seleções Precisas

Retangular, Elíptica, Lasso, Varinha Mágica — com marching ants, máscaras de seleção, feather e modos de adição/subtração.

Filtros em Tempo Real

Blur, Sharpen, Edge Detect, Pixelate, Posterize, Grayscale, Invert, Sepia. Ajustes não-destrutivos com camadas de ajuste.

Pen / Bezier Completo

Curvas cúbicas com handles de controle. Adição/remoção de pontos âncora, path editing — como no Illustrator ou Affinity Designer.

Undo/Redo Ilimitado

100 níveis de histórico com preview visual no History Panel. Snapshots completos para retroceder qualquer operação.

Instalação

Pronto em segundos

Python, PyQt5 e numpy são as únicas dependências. 3 comandos e você está editando.

▶

Python (PyQt5)

git clone https://github.com/TyagoAlves/reverseaffinite.git
cd reverseaffinite
pip install PyQt5 numpy
python3 main.py
✓ reverseaffinite Photo — pronto!
⚡

C++ Engine (Qt5/OpenGL)

cd reverseaffinite/cpp_editor
chmod +x build.sh
./build.sh # auto-detecta distro
✓ C++ engine compiled — ~304 KB binary

Auto-detecta: apt (Debian/Ubuntu), dnf (Fedora), pacman (Arch), zypper (openSUSE), apk (Alpine), xbps (Void), emerge (Gentoo).

⚅

Windows (PowerShell)

cd reverseaffinite\cpp_editor
powershell -ExecutionPolicy Bypass .\build_windows.ps1
✓ reverseaffinite.exe built

Requer CMake + Qt5 + Visual Studio Build Tools. Detecta GPU NVIDIA automaticamente.

Arquitetura

Dual Engine

Python/PyQt5 para prototipação rápida, C++/Qt5/OpenGL para performance de produção.

🐍 Python Engine

▸ Interface completa com PyQt5
▸ 15+ ferramentas com atalhos profissionais
▸ Blend modes via NumPy (SIMD-ready)
▸ Histórico de 100 níveis com snapshots
▸ 150+ testes automatizados

⚡ C++ Engine

▸ Pipeline de renderização OpenGL
▸ Pixel ops com SSE2 (128-bit SIMD)
▸ Qt5 com OpenGL canvas
▸ Binário ~104 KB otimizado
▸ Build via aqtinstall + CMake
