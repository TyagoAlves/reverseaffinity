Início Rápido

Em 3 passos você tem o reverseaffinite rodando no Linux:

1
Clone e instale dependências
git clone https://github.com/TyagoAlves/reverseaffinite.git
cd reverseaffinite
pip install PyQt5 numpy
2
Inicie o editor
python3 main.py

A janela do editor abrirá com layout profissional: paleta de ferramentas à esquerda, canvas central, painéis à direita.

3
Crie ou abra uma imagem

Arquivo → Novo (Ctrl+N) para criar uma tela em branco ou Arquivo → Abrir (Ctrl+O) para editar uma imagem existente. Formatos suportados: PNG, JPG, BMP, TIFF, GIF.

Ferramentas

O reverseaffinite possui 15+ ferramentas profissionais organizadas na paleta vertical à esquerda. Cada ferramenta tem atalho de tecla única.

V Move

Arrasta camadas e seleções pelo canvas. Com Shift mantém proporção. Nós de transformação nos cantos para redimensionar e rotacionar.

M Seleção Retangular

Desenha seleções retangulares. Com Shift cria quadrados perfeitos.

L Lasso

Seleção à mão livre. Ideal para contornos orgânicos e recortes precisos.

W Varinha Mágica

Seleciona áreas por similaridade de cor. Tolerância ajustável no painel de opções.

B Brush

Pincel com tamanho, opacidade e flow ajustáveis. Suporta tablet com sensibilidade à pressão.

E Eraser

Apaga pixels da camada ativa. Em máscaras de camada, revela ou oculta áreas.

P Pen

Curvas Bezier com handles de controle. Adiciona/remove pontos âncora. Cria paths que podem ser convertidos em seleções ou shapes.

G Gradient

Editor de gradientes com múltiplos stops, tipos (linear, radial, angular, reflected) e cores personalizáveis.

S Shape

Retângulos, elipses, triângulos e polígonos com preenchimento sólido ou gradiente.

T Text

Texto com fonte, tamanho, cor e alinhamento. Edição inline no canvas.

C Clone Stamp

Clona pixels de uma região para outra. Alt+clique define origem.

H Healing Brush

Corrige imperfeições combinando textura e iluminação da área de origem com o destino.

O Crop

Corta a imagem para o tamanho desejado. Preview com grade de terços.

I Color Picker

Seleciona cores do canvas. Define cor de frente e fundo.

Z Zoom

Aproxima e afasta o canvas. Clique para ampliar, Alt+clique para reduzir.

Camadas & Mesclagem

Sistema completo de camadas com 12 modos de mesclagem, máscaras e grupos.

Painel de Camadas

Listagem: Cada camada aparece com thumbnail, nome, visibilidade (olho), opacidade e modo de mesclagem.

Ordem: Arraste para reordenar. Camadas no topo aparecem na frente.

Grupos: Pastas que agrupam camadas. Aplique opacidade/mesclagem ao grupo inteiro.

Máscaras: Adicione máscara de camada. Pinte de preto para ocultar, branco para revelar.

Lock: Trave camada para evitar edição acidental.

Modos de Mesclagem (Blend Modes)

Normal

Camada cobre a de baixo. Opacidade controla a transparência.

Multiply

Escurece a imagem combinando cores. Branco não tem efeito.

Screen

Clareia a imagem. Preto não tem efeito.

Overlay

Combina Multiply e Screen. Aumenta contraste.

Darken

Mantém os pixels mais escuros de cada canal.

Lighten

Mantém os pixels mais claros de cada canal.

Difference

Subtrai cores. Preto = sem efeito, branco = inverte.

Add

Soma canais de cor. Cria efeitos de luz e brilho.

Subtract

Subtrai canais de cor. Escurece a imagem.

Dodge

Clareia baseado no tom da camada. Efeito forte.

Burn

Escurece baseado no tom da camada. Efeito forte.

Soft Light

Similar ao Overlay mas mais suave. Bom para texturas.

Filtros & Ajustes

Galeria completa de filtros e camadas de ajuste não-destrutivas.

Filtros Destrutivos (Menu Filter)

Blur

Desfoque gaussiano com raio ajustável. Suaviza pixels.

Sharpen

Aumenta o contraste entre pixels vizinhos. Nitidez.

Edge Detect

Detecta bordas com filtro Sobel. Resultado em preto e branco.

Pixelate

Efeito mosaico com tamanho de bloco ajustável.

Posterize

Reduz número de cores por canal. Efeito cartaz.

Grayscale

Remove saturação. Converte para tons de cinza.

Invert

Inverte todas as cores. Negativo da imagem.

Sepia

Tom vintage sépia. Efeito fotografia antiga.

Camadas de Ajuste (Não-destrutivas)

Camadas de ajuste aplicam modificações na pilha de camadas abaixo sem alterar os pixels originais.

▸ Brightness/Contrast: Ajusta luminosidade e contraste geral.

▸ Hue/Saturation: Matiz, saturação e luminosidade.

▸ Levels: Controle de tons escuros, médios e claros com histograma.

▸ Curves: Curvas de tom com pontos de controle.

Seleções

Sistema completo de seleções com "marching ants", máscaras e modos booleanos.

Tipos de Seleção

Retangular (M): Arraste para criar. Shift = quadrado.

Elíptica: Mesmo comportamento da retangular, mas elíptica.

Lasso (L): Desenho livre. Solte o mouse para fechar a seleção.

Varinha (W): Clique em uma área. Tolerância no painel de opções.

Modos Booleanos

Nova

Substitui seleção atual

Adicionar

Soma à seleção existente

Subtrair

Remove da seleção

Intersectar

Apenas área comum

Operações

Selecionar Tudo: Ctrl+A

Desselecionar: Ctrl+D

Inverter Seleção: Ctrl+Shift+I

Suavizar Bordas: Feather no painel de opções antes de criar.

Pen & Paths

Ferramenta Pen com curvas Bezier cúbicas — como no Illustrator ou Affinity Designer.

Como Usar

1. Selecione a ferramenta Pen (P).

2. Clique no canvas para adicionar pontos âncora.

3. Arraste ao clicar para criar handles de curva (Bezier).

4. Clique no primeiro ponto para fechar o path.

5. Clique com botão direito em um ponto para removê-lo.

Ações

Converter em Seleção: Botão direito → "To Selection" ou Ctrl+Enter.

Preencher Path: Preenche com a cor de frente.

Contornar Path: Desenha traçado com o brush atual.

Editor de Gradientes

Editor profissional com múltiplos stops, tipos de gradiente e cores personalizáveis.

Tipos de Gradiente

Linear

Transição em linha reta

Radial

Círculo concêntrico

Angular

360 graus

Reflected

Espelhado

Controles

▸ Adicione stops clicando na barra de gradiente.

▸ Arraste stops para reposicionar.

▸ Clique duas vezes em um stop para escolher cor.

▸ Arraste para cima/para fora para remover um stop.

▸ Controle de suavidade entre stops (midpoint).

Engine C++ (Em Desenvolvimento)

Motor de renderização nativo C++ com Qt5 e OpenGL para performance de produção.

⚡ Pipeline OpenGL

  • Canvas acelerado por GPU
  • Zoom e pan suaves (60fps)
  • Renderização de camadas via shaders
  • Blend modes em GLSL

⚡ SIMD Pixel Ops

  • Operações SSE2 (128-bit) em pixels RGBA
  • Blend modes sem分支预测
  • Fallback escalar para CPUs sem SSE2
  • Binário otimizado ~104 KB

Build

cd cpp_editor
chmod +x build.sh
./build.sh

Requer Qt5 (download automático via aqtinstall) e CMake.

Perguntas Frequentes

As dúvidas mais comuns sobre o reverseaffinite Photo.

O reverseaffinite funciona no Windows ou macOS?

Atualmente o foco é Linux, mas como é Python/PyQt5, funciona em Windows e macOS também com pequenos ajustes.

Preciso de placa de vídeo?

Não. O engine Python usa CPU (NumPy). O engine C++ usa OpenGL e se beneficia de GPU, mas tem fallback CPU.

O projeto tem suporte a plugins?

Ainda não, mas a arquitetura de ferramentas é extensível por herança da classe base Tool. Futuramente teremos sistema de plugins Python.

Qual a diferença para o GIMP?

O reverseaffinite busca o fluxo de trabalho do Affinity Photo / Photoshop — layout profissional com paleta à esquerda, painéis à direita, atalhos de tecla única. É mais leve e focado em usabilidade moderna.

Posso contribuir?

Sim! O projeto é 100% open source (MIT). Faça fork no GitHub, implemente melhorias e abra um Pull Request.
