Instalação

Pronto em segundos

Python, PyQt5 e numpy são as únicas dependências.

▶ Python (PyQt5)

$ git clone https://github.com/TyagoAlves/reverseaffinite.git
$ cd reverseaffinite
$ pip install PyQt5 numpy
$ python3 main.py
✓ reverseaffinite Photo — pronto!

⚡ C++ Engine (Qt5/OpenGL)

$ cd reverseaffinite/cpp_editor
$ chmod +x build.sh
$ ./build.sh
✓ C++ engine compiled — ~104 KB binary

Requer Qt5 (download automático via aqtinstall).

Recursos

Tudo que você precisa
para editar como um profissional

15+ Ferramentas Profissionais

Move, Brush, Eraser, Pen (Bezier), Gradient, Shape, Text, Clone Stamp, Healing Brush, Crop, e mais. Cada ferramenta com atalho de tecla única.

Camadas + 12 Blend Modes

Sistema completo de camadas com modos de mesclagem (Normal, Multiply, Screen, Overlay, etc.), opacidade, visibilidade, máscaras e grupos.

Seleções Precisas

Retangular, Elíptica, Lasso, Varinha Mágica — com "marching ants", máscaras de seleção e modos de adição/subtração.

Filtros em Tempo Real

Galeria de filtros: Blur, Sharpen, Edge Detect, Pixelate, Posterize, Grayscale, Invert, Sepia. Ajustes não-destrutivos com camadas de ajuste.

Pen / Bezier

Curvas cúbicas com handles de controle. Adição/remoção de pontos âncora, path editing completo — como no Illustrator ou Affinity Designer.

Undo/Redo Ilimitado

100 níveis de histórico com preview visual no History Panel. Snapshots completos das camadas para retroceder qualquer operação.

Arquitetura

Dual Engine

Python/PyQt5 para prototipação rápida, C++/Qt5/OpenGL para performance de produção.

🐍 Python Engine

  • ▸ Interface completa com PyQt5
  • ▸ 15+ ferramentas com atalhos profissionais
  • ▸ Blend modes via NumPy (SIMD-ready)
  • ▸ Histórico de 100 níveis com snapshots
  • ▸ 136 testes automatizados

⚡ C++ Engine

  • ▸ Pipeline de renderização OpenGL
  • ▸ Pixel ops com SSE2 (128-bit SIMD)
  • ▸ Qt5 com OpenGL canvas
  • ▸ Binário ~104 KB otimizado
  • ▸ Build via aqtinstall + CMake
