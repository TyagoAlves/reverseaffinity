1. Introdução

reverseaffinite Photo é um editor de imagens profissional desenvolvido em Python com PyQt5, combinando a produtividade do Affinity Photo, o poder do Adobe Photoshop e a fluência do DaVinci Resolve em uma experiência nativa para Linux.

Principais características

  • 15+ ferramentas com atalhos profissionais de teclado
  • Camadas com 12 modos de mesclagem, opacidade e máscaras
  • Seleções retangular, elíptica, laço e varinha mágica
  • Galeria de filtros com blur, sharpen, detecção de bordas e mais
  • Camadas de ajuste não-destrutivas (Brilho/Contraste, HSL, Níveis)
  • Ferramenta Caneta com curvas Bezier e handles de controle
  • Pincel de cura com casamento automático de cor
  • Gradientes com editor visual completo
  • Guias, régua e snapping para layout de precisão
  • Undo/Redo de 100 níveis com painel de histórico visual
  • Modo escuro profissional incluso

2. Início Rápido

Requisitos

  • Python 3.8+
  • PyQt5 5.15+
  • NumPy

Instalação

git clone https://github.com/TyagoAlves/reverseaffinite.git
cd reverseaffinite
pip install PyQt5 numpy
python3 main.py

Ao iniciar, uma janela será aberta com uma tela em branco (800×600). A interface segue o layout profissional: paleta de ferramentas à esquerda, canvas ao centro e painéis à direita.

3. Tour da Interface

A interface é dividida em quatro áreas principais:

  • Barra de Menu (topo): File, Edit, Image, Layer, Filter, View, Help
  • Paleta de Ferramentas (esquerda): 15+ ferramentas com atalho de tecla única
  • Canvas (centro): Área de edição principal com zoom e pan
  • Painéis (direita): Color, Layers, Navigator, History

5. Sistema de Camadas

O sistema de camadas do reverseaffinite oferece:

  • Adicionar/remover/renomear camadas
  • 12 modos de mesclagem: Normal, Multiply, Screen, Overlay, Darken, Lighten, Color Dodge, Color Burn, Hard Light, Soft Light, Difference, Exclusion
  • Opacidade ajustável (0–100%)
  • Visibilidade on/off
  • Máscaras de camada
  • Grupos de camadas
  • Camadas de ajuste não-destrutivas
  • Duplicar e mesclar camadas
  • Batch export de camadas individuais

6. Guia de Seleções

As ferramentas de seleção seguem o padrão da indústria:

  • Rect Select / Ellipse Select: Arraste para criar. Segure Shift para círculo/quadrado perfeito.
  • Lasso: Clique e arraste para seleção à mão livre.
  • Magic Wand: Clique em uma área para selecionar pixels similares. Ajuste a tolerância na barra de opções.

Modos de interação:

  • Nova seleção: clica e arrasta normalmente
  • Adicionar: segura Shift enquanto seleciona
  • Subtrair: segura Alt enquanto seleciona

7. Filtros e Ajustes

Galeria de Filtros

Acessível pelo menu Filter → Filter Gallery ou Ctrl+F:

  • Blur: Desfoque gaussiano
  • Sharpen: Nitidez
  • Edge Detect: Detecção de bordas
  • Pixelate: Pixelização
  • Posterize: Posterização
  • Grayscale: Escala de cinza
  • Invert: Inverter cores
  • Sepia: Tom sépia

Camadas de Ajuste

Acessível pelo menu Layer → New Adjustment Layer:

  • Brilho/Contraste: Ajuste de luminosidade e contraste
  • HSL: Matiz, saturação e luminosidade
  • Níveis: Controle de tons escuros, médios e claros
